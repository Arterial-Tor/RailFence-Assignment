# RailFence-Assignment

***** RailFence Encryptor 3000 v1.0 *****

************************* MAIN PROGRAMME FEATURES *************************

On running the program user will be presented with a simple menu, shown below.

######################################
    RailFence Encryptor 3000 v1.0
######################################
1) Select File
2) Input Plain Text
3) Enter RailFence Key
4) Encrypt
5) Decrypt
6) Quit
Select an Option [1 - 6]>

Users can manually select an option to perform an operation witin the program.

The basic function of this application will be to encrypt/decrypt files using a basic railfence cipher.
Users are required to provide three key pieces of information to run the RailFence Cipher.

1) Text that is to be encrypted or decrypted (file or manual entry)
2) Encryption Key > 1
3) Key Offset

The encryption key and offset are vital for encryption and successful decryption of the chosen text.
